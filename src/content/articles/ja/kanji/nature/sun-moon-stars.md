### 太陽

#### 【一字】

| 日  | ひ     |     |
| --- | ------ | --- |
| 陽  |
| 曦  | ひかり |
| 烏  | からす |     |

太陽には烏がいるという伝説から、烏は太陽の例えやシンボルに使われる。

#### 【熟語】

日月に私照無し日暈陽光陽暦金烏八咫烏

---

### 月

#### 【一字】

| 月  | つき     |
| --- | -------- |
| 蟾  |
| 朏  | みかづき |
| 朦  | おぼろ   |
| 朧  |
| 姮  | コウ     |
| 娥  | ガ       |
| 兎  | うさぎ   |

月には兎がいるという伝説から、兎は月の例えやシンボルに使われる。 蟾(つき)は西王母の仙薬を盗んで月へ逃げた姮娥がヒキガエルになったという古代中国の伝説による。

#### 【熟語】

蟾月蟾光蟾円姮娥
