# 行動規範

Atlasezは、学年・専攻・経験・背景にかかわらず、誰もが安心して学び・貢献できる場を目指します。

## 期待される行動

- 相手を尊重した言葉づかいをする
- 初学者の質問・不慣れなPRを歓迎し、建設的にレビューする
- 間違いの指摘は内容に向け、人に向けない
- 多様な視点・学び方を尊重する

## 許容されない行動

- ハラスメント、差別的言動、個人攻撃
- 他者の個人情報の無断公開
- 学習コミュニティの目的にそぐわない宣伝・スパム

## 通報

問題を見つけた場合は atlasez.contact@gmail.com へ連絡してください。
運営が確認し、注意・コントリビューション制限などの対応を行います。

この行動規範は [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 を参考にしています。
